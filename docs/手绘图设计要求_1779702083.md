# 手绘图设计要求

#### You:
生成一张手绘图，淡蓝色科技感背景，搭配亮黄色和粉色等彩色点缀，线条简洁、扁平化、画面干净。画面中心是一个巨大的审核工作台，前景和背景中充满了很多活动卡片，像看板上一排排的活动卡片或小窗口，每一张上面都有「活动名称 + 配置信息」的简化展示，体现“活动数量很多、配置各不相同”。画面右侧坐着一个可爱的 LLM 小机器人审核员，机器人有显示屏脸和小天线，表情专注但友好，一只机械手在飞快地翻阅活动配置卡片，另一只手在界面上点击“通过 / 拒绝”按钮，屏幕上出现绿色对勾和红色叉号，表现出对配置合理性的对比和判断。机器人身后有类似大脑或芯片的发光图标，表示智能大模型在思考，周围有流动的数据流线条，突出“自动审核、快速决策”的感觉。整体风格偏可爱动漫、圆角图形、2～3 像素细线条、扁平扁桃形图标，既专业又有趣，适合作为“多活动智能审核”场景的产品插画
